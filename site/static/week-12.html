Phase 6

Week 12. Demo script + capstone charter.

Learn chapter: Prototypes — concepts, videos, and links. Study alongside or before practice.

Problem statement

You have 8 minutes with VP-level stakeholders. Script the demo; start the Insight copilot capstone per master plan.

Outcomes

  • 5-minute demo script with fallback lines.
  • Capstone charter: problem, users, scope, non-goals, success metrics.

Deliverables

  • demo_script.md (timed reading ~8 min aloud).
  • capstone_charter.md (1–2 pages).

Links

  • Capstone
  • [ ] Re-read Capstone section in master plan.
  • [ ] Optional: one post on offline vs online eval.

Time: ~2 h.

Task A — Demo script

demo_script.md: sections Setup → Question 1 (insight from DB) → Question 2 (doc/policy) → Failure fallback → Q&A. Include exact opening line and exact line if API fails.

Task B — Capstone charter

capstone_charter.md must define:

  • Problem & user
  • In-scope / out-of-scope
  • Metrics (quality + ops + cost)
  • Risks (safety, wrong SQL, stale docs)
  • Week 13–14 milestones

Task C — Dry run

Practice aloud once; record actual duration; trim script if > 8 min.
