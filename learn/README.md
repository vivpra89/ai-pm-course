# Learn hub — concepts first, practice weekly

This hub is **topic-organized** for a smooth learning path: read and watch in order, take notes, then go to **Weekly practice** for hands-on work.

| Track | What it is |
|-------|------------|
| **Learn** (this section) | Concepts, curated articles & videos, mental models — **not** tied to calendar pace |
| **Weekly practice** (`course-weeks/week-XX`) | Readings, labs, exams — **one week at a time** |
| **Practice environment** | SQLite CloudNote DB, corpus, scripts — shared lab bench |

**Suggested flow**

1. Work through the **Learn chapters** below in order (or jump to what your job needs next).
2. Use **Weekly practice** for structured rhythm: each week has Overview → Readings → Practice → Exam.
3. Keep artifacts in `my-submissions/week-XX/`.
