# Agents, reviews, and launch

*Estimated study: 4–6 hours · aligns with Weeks 13–14 (final review).*

---

## Outcomes

- **Agent** = model + **tool loop** + **stop conditions** + **human gates** — not magic.  
- **Red team** / **safety review** before broad rollout.  
- **Launch checklist**: monitoring, rollback, incident comms, model card summary.  

---

## Topic map

| Topic | Core idea |
|-------|-----------|
| **Agent loop** | Plan → act → observe — budget steps & tokens. |
| **Human-in-the-loop** | Approvals for writes, customer-visible messages. |
| **Eval in prod** | Shadow mode, canary, drift alerts on quality metrics. |
| **Incident response** | Playbook when model output harms or leaks. |
